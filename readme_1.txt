                                            Документация к проекту

Название проекта: Менеджер книг
Описание: Менеджер книг - это программа, которая позволяет пользователям смотреть добавленные книги, искать добавленные книги, удалять и добовлять книги а также узнать подробнее о книге.
Программа использует базу данных SQLite для хранения книг.


                                                Функции программы

Просмотр списка книг: Пользователь может просмотреть список всех сохраненных книг или выбрать жанр и просмотреть книги с данным жанром.
Поиск книг: Пользователь может выполнить поиск книг по названию, автору или названию и автору.
Удаление книги: Пользователь может удалить книгу из базы данных, указав ее точное название.
Добавление книги: У пользователя попросит  авт, после чего заметка будет сохранена в базе данных.
Узнать подробнее о книге: Пользователь может узнать автора, описание и жанр указав точное название книги

Инструкция по запуску и тестированию программы:

Установите Python версии 3.x на ваш компьютер, если он ещё не установлен.
Скачайте файл с исходным кодом программы и сохраните его на вашем компьютере.
Установите модуль SQLite3, если он ещё не установлен, с помощью команды pip install pysqlite3.
Откройте командную строку или терминал и перейдите в каталог, содержащий файл с исходным кодом программы.
Запустите программу, выполнив команду python <название_файла>.py, где <название_файла> - имя файла с исходным кодом программы.
Взаимодействуйте с программой, следуя инструкциям в меню.
